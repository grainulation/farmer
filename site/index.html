How to use farmer
1

Start the server

Launch Farmer from your project directory. It prints a token-authenticated URL.

$ npx @grainulator/farmer start # Dashboard: http://localhost:9090?token=abc123
2

Configure hooks

Point your Claude Code agent at Farmer's hook endpoint in your settings.

// .claude/settings.json { "hooks": { "PreToolUse": [{ "type": "command", "command": "curl -s POST ..." }] } }
3

Open the dashboard

Open the token URL in your browser or phone. Permission cards appear in real time as your agent works.

# Open the URL from step 1 # Permission cards stream live # Approve or deny with one tap

Key features

Control without context-switching

Zero npm dependencies. Node built-ins only. No framework, no build step.

Token Authentication

Every session gets a unique token. No token, no access. The URL printed at startup is the only way in.

WebSocket Live Updates

Permission cards stream to your browser instantly over WebSocket. No polling, no lag.

Multi-Session

Run multiple AI agents at once. Each gets its own session in the sidebar. Switch with a click.

Trust Tiers

Paranoid, Standard, or Autonomous. Control how much your agent can do without asking.

Audit Logging

Every approve, deny, and auto-decision is logged with timestamps. Full accountability.

Mobile Ready

Responsive layout works on your phone. Approve from the couch, the bus, or a meeting.
