Setup
Three steps. Under a minute.
No config files, no build step, no dependencies to install. One command gets you a running dashboard.
1

Start the server

Launch Farmer from your project directory. It prints a token-authenticated URL to your terminal.

# Start with defaults (port 9090)
npx @grainulator/farmer start

# Or pick a port
npx @grainulator/farmer start --port 8080
2

Configure your agent's hooks

Point your agent at Farmer's hook endpoints. Here is the Claude Code configuration:

// .claude/settings.json
{
"hooks": {
"PreToolUse": [{
"type": "command",
"command": "curl -s -X POST http://localhost:9090/hooks/permission -H 'Content-Type: application/json' -d \"$TOOL_INPUT\""
}]
}
}
3

Open the dashboard

The token URL printed at startup opens the dashboard. Bookmark it, send it to your phone, or keep it in a browser tab.

# Printed when server starts:
Dashboard: http://localhost:9090?token=abc123

# That's it. Permission cards appear
# in real time as your agent works.

Adapters
Your agent. Your rules.
Farmer ships with a Claude Code adapter. Supporting a new agent is one class -- extend BaseAdapter, map request/response formats, and you're done.
import { BaseAdapter } from
'@grainulator/farmer/lib/adapters/base.js';

class MyAdapter extends BaseAdapter {
get name() { return 'My Agent'; }
parseRequest(body) { /* ... */ }
formatResponse(decision) { /* ... */ }
}

Features
Everything you need. Nothing you don't.
Zero npm dependencies. Node built-ins only. No framework, no build step, no bloat.

Token Authentication

Every session gets a unique token. No token, no access. The URL printed at startup is the only way in.

WebSocket Live Updates

Permission cards, activity events, and session status stream to your browser instantly. No polling.

Multi-Session

Run multiple AI agents at once. Each gets its own session in the sidebar. Switch between them with a click.

Trust Tiers

Paranoid, Standard, or Autonomous. Control how much your agent can do without asking. Change it mid-session.

Audit Logging

Every approve, deny, and auto-decision is logged with timestamps. Full accountability for every tool call.

Mobile Support

Responsive layout works on your phone. Approve a tool call from the couch, the bus, or a meeting.
